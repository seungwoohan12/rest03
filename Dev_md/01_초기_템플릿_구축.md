# 01. 초기 템플릿 구축

작업일: 2026-06-08

## 목표
chinhung.co.kr(진흥기업) 사이트의 디자인·구성을 참고하여 건설사 회사 홈페이지를 React 템플릿으로 구축한다. 모든 이미지는 플레이스홀더로 표기한다.

## 참고 소스
- 제공받은 JSX/Tailwind 소스 3종: 메인 / 사업소개(토목) / 지속가능경영(품질경영)
- 참고 사이트: https://www.chinhung.co.kr/

## 기술 선택
- React 18 + Vite 5 + Tailwind 3 + React Router 6
- GitHub Pages 정적 호스팅 호환을 위해 **HashRouter** 사용
- Vite `base: '/rest03/'`

## 구현 범위
| 영역 | 구현 내용 |
| --- | --- |
| 공통 | Header(데스크탑 메가메뉴/모바일 패널), Footer(주소·Family site·정책링크), 맨위로 버튼, 라우트 전환 스크롤 초기화 |
| 메인 | 히어로 슬라이더(자동전환·좌우·인디케이터), 회사소개 링크, Our Business 카드(hover 설명), Sustainability 배너, More to Discover 공지 |
| 사업소개 | sticky 탭 네비 + 페이지헤드 + 소분류 탭버튼 + 실적 4열 그리드 + 페이지네이션(정적) |
| 지속가능경영 | 품질경영 선언·방침 카드 3개·인증현황(ISO 9001/14001), 그 외 탭은 공통 placeholder 본문 |
| 회사소개 | CEO 인사말, 연혁, 비전/브랜드 placeholder |
| 기타 | 투자정보/고객센터/인재채용 등 SimplePage |

## 데이터 분리
- `src/data/site.js` 한 곳에서 회사정보·GNB·사업카드·사업메타·실적·공지를 관리 → 템플릿 재사용성 확보.
- 토목/도로포장 실적은 소스의 실제 데이터를 반영, 나머지 카테고리는 데모 데이터 자동 생성.

## 이미지 플레이스홀더
- `Placeholder` 컴포넌트: 라벨 + 비율(aspect-ratio) + 사선 패턴 표시. 실제 운영 시 `<img>` 로 교체.

## 검증
- `npm run build` 성공 (46 modules, 빌드 ~1.5s)
- `npm run dev` 부팅 및 HTTP 200 확인

## 배포
- `.github/workflows/deploy.yml`: main push 시 빌드 → GitHub Pages 자동 배포
- 배포 URL: https://aebonlee.github.io/rest03/
- ⚠️ 최초 1회 GitHub 저장소 Settings → Pages → Source 를 **GitHub Actions** 로 전환 필요
