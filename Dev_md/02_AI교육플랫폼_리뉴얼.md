# 02. AI 교육 플랫폼 리뉴얼

작업일: 2026-06-10

## 목표

기존 건설사(진흥기업) 회사 사이트 템플릿을 **온라인 동영상 콘텐츠 기반 AI 교육 회사 사이트**로 전면 재구성한다.

- 브랜드명: **AILearn** (에이아이런) — AI 기초·AI 리터러시 온라인 강의 플랫폼
- 주 기능: YouTube 동영상을 주제별로 분류해 2×3 그리드로 제공, 페이지네이션·모달 재생
- 다크모드 & 라이트모드 지원
- 모바일 최적화

## 다크/라이트 모드

- Tailwind `darkMode: 'class'` 전략 사용
- `src/context/ThemeContext.jsx` 신규 생성
  - 초기화: `localStorage('theme')` → 없으면 `prefers-color-scheme` OS 설정
  - `<html>` 요소에 `.dark` 클래스 토글
  - `document.body.style.backgroundColor` 즉시 적용 (색상 깜빡임 방지)
- `index.html` 인라인 스크립트로 렌더 전 다크모드 적용 (FOUC 완전 차단)
- 헤더 우측 ☀️/🌙 토글 버튼, 모바일 패널에도 동일 토글 제공

## 사이트 구조

```
/                        홈
├── /about/:tab          회사소개 (intro / philosophy / instructors / history)
├── /videos/ai-related   AI 관련 영상 (sub: ai-basics / ai-tools / ai-trends)
├── /videos/ai-literacy  AI 리터러시 (sub: ai-understanding / data-thinking / ai-ethics)
├── /contact             문의하기
├── /notice              공지사항 (SimplePage)
└── /privacy 등          기타 SimplePage
```

라우터: HashRouter (GitHub Pages 정적 호스팅 호환)

## 영상 페이지 핵심 설계

### 그리드: 2열 × 3행 = 페이지당 6개

```
┌─────────────┐ ┌─────────────┐
│  Video 1    │ │  Video 2    │
└─────────────┘ └─────────────┘
┌─────────────┐ ┌─────────────┐
│  Video 3    │ │  Video 4    │
└─────────────┘ └─────────────┘
┌─────────────┐ ┌─────────────┐
│  Video 5    │ │  Video 6    │
└─────────────┘ └─────────────┘
         ‹ 1  2  3 ›
```

- 모바일(< sm): 1열
- 태블릿 이상: 2열 (sm: 이상)

### 주제별 메뉴 구조

**대분류 탭** (sticky 헤더 하단)
- `AI 관련 영상` → `/videos/ai-related`
- `AI 리터러시` → `/videos/ai-literacy`

**소분류 필터** (URL ?sub= 파라미터)
- AI 관련: 전체 / AI 기초 / AI 도구 활용 / AI 트렌드
- AI 리터러시: 전체 / AI 이해 / 데이터 사고 / AI 윤리

### YouTube 임베드 모달

- 썸네일: `https://img.youtube.com/vi/{ID}/hqdefault.jpg`
- 로드 실패 시 그라디언트 폴백 (제목·카테고리 표시)
- 클릭 → `<iframe>` 모달 (`autoplay=1&rel=0&modestbranding=1`)
- ESC 키 또는 배경 클릭으로 닫기
- 모달 열림 시 `body overflow: hidden`

### 영상 ID 교체 방법

```js
// src/data/site.js  → videos 배열
{
  youtubeId: 'ABC123XYZ',  // ← 여기만 교체 (링크 공개 영상 권장)
  title: '영상 제목',
  ...
}
```

## 모바일 최적화 포인트

- Header: `lg:` 미만 햄버거 메뉴 → 슬라이드 오버레이 패널
- Hero: `min-h-[90vh]` + `text-5xl md:text-7xl lg:text-8xl` 반응형 폰트
- Video grid: `grid-cols-1 sm:grid-cols-2`
- Footer: `flex-col lg:flex-row`, 메뉴 `grid-cols-2 md:grid-cols-5`
- Sticky 탭: `overflow-x-auto`로 작은 화면에서 가로 스크롤
