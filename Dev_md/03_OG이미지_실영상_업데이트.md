## 1. OG 이미지 생성 (sharp + SVG)

### 기술 선택 이유

| 방법 | 채택 여부 | 이유 |
|------|-----------|------|
| sharp + SVG | ✅ 채택 | Node.js 내장, SVG → PNG 고품질 변환, librsvg 기반 |
| node-canvas | ❌ | 네이티브 빌드 의존성 복잡 (Windows 환경) |
| Puppeteer (HTML→PNG) | ❌ | Chromium 번들 부피 (~300MB) |
| 외부 API (Vercel OG 등) | ❌ | GitHub Pages 정적 환경 부적합 |

### 구현

```
scripts/generate-og.mjs
├── SVG 템플릿 (1200 x 630 px)
│   ├── 배경 그라디언트: Deep Navy → Dark Blue (#0B1D3A → #1E3A8A)
│   ├── 장식 원: Royal Blue / Sky Cyan 반투명
│   ├── 좌측 세로 포인트 바: Royal → Sky 그라디언트
│   ├── AI 아이콘 박스 (Royal Blue)
│   ├── 회사명 AILearn
│   ├── 태그라인 (영문)
│   └── 5색 팔레트 도트 (하단)
└── sharp().png().toFile('public/og-image.png')
```

실행 명령: `node scripts/generate-og.mjs`

### OG 이미지 규격

| 항목 | 값 |
|------|-----|
| 크기 | 1200 × 630 px |
| 형식 | PNG (품질 95, 압축 6) |
| 경로 | `public/og-image.png` |
| 배포 URL | `https://seungwoohan12.github.io/rest03/og-image.png` |

## 4. YouTube Shorts 대응

Shorts 영상은 세로형(9:16) 콘텐츠이므로 별도 처리:

| 항목 | 일반 영상 | Shorts |
|------|----------|--------|
| 썸네일 비율 | `aspect-video` (16:9) | `aspect-[9/16]` + 최대 폭 제한 |
| 모달 embed | `aspect-video` | `aspect-[9/16] max-w-sm` |
| Shorts 뱃지 | 없음 | 썸네일 좌상단 표시 |
| `isShort` 플래그 | false | true |

embed URL은 동일: `https://www.youtube.com/embed/{ID}?autoplay=1`
