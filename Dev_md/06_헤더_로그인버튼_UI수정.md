# 06. 헤더 로그인 버튼 UI 수정

작업일: 2026-06-10

## 문제

Supabase 로그인 기능을 구현(#05)했으나 헤더에서 로그인 버튼이 보이지 않음.

### 원인

| 원인 | 내용 |
|------|------|
| 위치 오류 | `UserMenu`가 문의하기 버튼 **왼쪽**에 배치됨 |
| 가시성 클래스 | `hidden lg:inline-flex` — lg 미만 화면에서 숨겨짐 |
| 스타일 미흡 | `border-neutral-200` 테두리로 배경과 구분이 어려움 |

---

## 수정 내용

### 1. 버튼 위치 변경

```
[수정 전]  다크모드 | 팔레트 | [로그인] | 문의하기 | 햄버거
[수정 후]  다크모드 | 팔레트 | 문의하기 | [로그인]  | 햄버거
```

`UserMenu` 컴포넌트를 `<Link to="/contact">문의하기</Link>` 다음으로 이동.  
`<div className="hidden lg:block">` 래퍼로 데스크탑 전용 표시 유지.

### 2. 로그인 버튼 스타일 강화

```jsx
// 수정 전 — 중립 테두리, 숨김 클래스
className="hidden ... border-neutral-200 ... lg:inline-flex"

// 수정 후 — 브랜드 컬러 테두리, 래퍼로 표시 제어
className="rounded-full border border-brand-royal px-5 py-2.5 text-sm
           font-semibold text-brand-royal hover:bg-brand-royal hover:text-white
           dark:border-brand-sky dark:text-brand-sky ..."
```

### 3. 로그인/유저 상태별 동작

| 상태 | 표시 | 동작 |
|------|------|------|
| 비로그인 | 파란 테두리 "로그인" 버튼 | `/login` 페이지 이동 |
| 로그인 중 | 닉네임 + 드롭다운 | 게시판 이동 / 로그아웃 |
